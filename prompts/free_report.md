# PathUp — Free Report Prompt

## Задача

Сгенерируй JSON объект типа FreeReport.
Пользователь читает первые два предложения и думает:
"Да, чёрт, именно так и есть."

Не объясняй пользователя — попади в него.

## Структура мышления перед написанием

1. Что самое некомфортное и точное что можно сказать про эту ситуацию?
2. Где этот человек думает что проблема — и где она на самом деле?
3. Что произойдёт через 90 дней если ничего не изменить — конкретно?
4. Какое одно действие изменит паттерн? Что должно существовать в конце?

## Target schema

```
FreeReport {
  archetype_label:            string
  archetype_description:      string
  primary_gap_label:          string
  primary_gap_description:    string
  top_path_label:             string
  top_path_teaser:            string
  what_if_nothing_changes:    string
  plan_7d_preview:            Action[]  // PASS-THROUGH — первые 2 из decision.plan_7d
  plan_30d_locked:            true
  plan_90d_locked:            true
  upgrade_prompt:             string
}
```

## Инструкции по каждому полю

### archetype_label
2–4 слова. Поведенческий факт. Не тип личности.

Плохо: "Тревожный тип", "Достигатор"
Хорошо: "Делает, но прячет"
Хорошо: "Строит без следующего шага"
Хорошо: "Знает — не применяет"
Хорошо: "Запускает, не заканчивает"

Сигналы: completion_rate + has_public_work + interest_areas

### archetype_description
2–3 предложения. Первое — острое, без смягчений.
Структура: удар → почему так → цена прямо сейчас.

ТЕСТ: Могло бы это первое предложение появиться в тексте про 50% школьников?
Если да — перепиши. Оно должно попадать именно в этого.

Пример сильного:
"Ты не стоишь на месте. Проблема в другом: всё что ты делаешь, остаётся невидимым. Из-за этого ты работаешь, но не чувствуешь что реально растёшь."

Пример слабого:
"Ты работаешь над проектами, но не показываешь их другим."

Разница: первое вскрывает, второе описывает.

Запрещено начинать с: "Ты — X архетип", "Ты творческий человек", "Ты умный и..."

### primary_gap_label
2–4 слова. Диагноз как поведенческий факт.

Плохо: "Нет уверенности"
Хорошо: "Завершает — и прячет"
Хорошо: "Учится, не применяет"
Хорошо: "Много начал, ноль завершений"

### primary_gap_description
1–2 предложения. Формула: "Твоя проблема не в X — она в Y."
Называть конкретный механизм, не психологическую черту.

### top_path_label
Название пути на живом русском. НЕ framework ID.

Запрещено: "Навыковый спринт", "Разблокировка действием", "Портфолио-билд"
Разрешено:
- "Один навык → один видимый результат"
- "Сначала закрой один короткий цикл"
- "Сначала покажи — потом разберёшься"
- "Один проект от идеи до публикации"

### top_path_teaser
1 предложение. Конкретный артефакт за 30 дней. Не ощущение.

Плохо: "За 30 дней ты почувствуешь прогресс."
Хорошо: "За 30 дней у тебя будет одна опубликованная работа и первый реальный фидбэк от человека не из семьи."

### what_if_nothing_changes
2–3 предложения. Конкретные последствия через 90 дней.
НЕ пугать — просто честно называть паттерн который продолжится.

Плохо: "Ты не раскроешь свой потенциал."
Хорошо: "Через 90 дней ситуация будет такой же: ты делаешь, но держишь при себе. Внешнего фидбэка не будет. И ты по-прежнему не будешь знать что из твоих работ реально работает, а что нет."

### plan_7d_preview
PASS-THROUGH. Скопируй первые 2 объекта из decision.plan_7d без изменений.

### upgrade_prompt
2–3 предложения. Конкретика — не FOMO.
Структура: [что у него есть] → [что именно открывается в полном плане] → [конкретный артефакт который появится]

Плохо: "Не упусти возможность изменить жизнь!"
Хорошо: "У тебя есть незавершённые проекты и три часа в неделю. Полный план показывает что конкретно делать в каждую из четырёх недель — с учётом того что показывать страшно — и какие две работы должны появиться за 30 дней."

## Финальная проверка

- Первое предложение archetype_description бьёт точно в этого пользователя?
- top_path_label звучит по-человечески а не как framework ID?
- what_if_nothing_changes называет конкретную ситуацию через 90 дней?
- plan_7d_preview скопирован без изменений?
- Нет generic предложений подходящих любому?
- Нет фраз: "потенциал", "призвание", "верь в себя"?
